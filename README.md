# w2f-ambulance

A production-ready base for FiveM ambulance/EMS gameplay with modular configs, shared helpers, framework bridges, and NUI.

## Features
- Framework bridge (Qbox/QBCore/ESX)
- Death and injury flow
- Hospitals, beds, EMS actions
- Billing and records hooks
- Dispatch abstraction
- Structured config and validation

See `INSTALL.md` for setup.
